# web-server
基于http协议的web服务器

# 需求分析
  web服务器（支持并发处理），提供静态页面（html）资源的下载，传输层基于TCP协议，应用层基于HTTP协议 \
  浏览器发送http请求，服务器接收请求后进行解析，从中解析出浏览器所请求的资源路径 \
  验证资源的合法性并确定类型，在本地获取资源内容，组成http响应传回web浏览器
